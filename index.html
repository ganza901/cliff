User Guide

Theme Settings

To change the theme of the application, follow these steps:

  1. Go to the Settings page.
  2. Find the Theme section.
  3. Select your preferred theme from the dropdown menu.
  4. Click Save Settings to apply the changes.

Available themes include:

  • Light Theme
  • Dark Theme
  • Colorful Theme

Language Settings

To change the language of the application, follow these steps:

  1. Go to the Settings page.
  2. Find the Language section.
  3. Select your preferred language from the dropdown menu.
  4. Click Save Settings to apply the changes.

Supported languages include:

  • English
  • Spanish
  • French
  • Kinyarwanda
